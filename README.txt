ASD lozabo MILANO - NUOVE PAGINE SITO WEB
===========================================

CONTENUTO DEL PACCHETTO:
------------------------

Questo pacchetto contiene 5 nuove pagine HTML con i relativi file CSS:

1. safeguarding-documenti.html + css/safeguarding-documenti.css
   - Pubblicazione documenti di safeguarding

2. newsletter-genitori.html + css/newsletter-genitori.css
   - Newsletter per i genitori degli atleti

3. newsletter-allenatori.html + css/newsletter-allenatori.css
   - Newsletter riservata ad allenatori e dirigenti

4. videopillole-atleti.html + css/videopillole-atleti.css
   - Brevi video monotematici per gli atleti

5. materiali-bambini.html + css/materiali-bambini.css
   - Materiali didattici per bambini e ragazzi


ISTRUZIONI PER L'INSTALLAZIONE:
--------------------------------

1. Creare una cartella "css" nella root del sito se non esiste già

2. Copiare tutti i file .html nella root del sito web

3. Copiare tutti i file .css nella cartella "css/"

4. Verificare che i percorsi dei CSS siano corretti:
   - Tutti i file HTML fanno riferimento a: href="css/nome-file.css"
   - Se la struttura del sito è diversa, aggiornare i percorsi di conseguenza


PERSONALIZZAZIONE:
------------------

Per personalizzare le pagine:

1. COLORI: Il colore principale rosso (#ff0000) può essere modificato
   nei file CSS cercando e sostituendo il valore esadecimale

2. CONTENUTI: Modificare i testi direttamente nei file HTML

3. LINK: Sostituire i link "#" con i percorsi reali dei documenti/video

4. IMMAGINI: Per le videopillole, sostituire i placeholder con immagini reali
   - Formato consigliato: 400x225px (16:9)

5. INTEGRAZIONE MENU: Aggiungere i link alle nuove pagine nel menu principale


FUNZIONALITÀ DA IMPLEMENTARE:
------------------------------

Le seguenti funzionalità richiedono JavaScript o backend:

1. Newsletter - Form di iscrizione (necessita integrazione email service)
2. Newsletter Allenatori - Sistema di login (necessita autenticazione)
3. Videopillole - Filtri per categoria (necessita JavaScript)
4. Materiali Bambini - Filtri per età (necessita JavaScript)
5. Download PDF - Link da collegare ai file reali


PROSSIMI PASSI CONSIGLIATI:
----------------------------

1. Iniziare con "Documenti Safeguarding" (più semplice)
2. Aggiungere "Newsletter Genitori"
3. Implementare "Materiali per Bambini e Ragazzi"
4. Successivamente "Videopillole per Atleti"
5. Infine "Newsletter Allenatori/Dirigenti" (richiede autenticazione)
